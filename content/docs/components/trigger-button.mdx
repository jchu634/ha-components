## About
This component is a button with an onClick which triggers a HA service (e.g. automations).\
This is intended for activating automations, but can trigger any HA service if the Domain and Service are specified.\
If you want to be able to see the current state of an entity, use [toggle](/docs/components/toggle) instead.

## API (Props)
<Callout >
    This only documents props added for triggering HA Services\
    For Button Styling Documentation look [here](https://ui.shadcn.com/docs/components/button)

</Callout>
### `entity`
e.g. `automation.open_curtains`
- A HA entity name

### `description` (Optional)
- Adds a title to the button

### `domain` (Optional)
- The Domain that the entity belongs to.
- Defaults to `automation`
- For more information on domains, see [here](https://www.home-assistant.io/docs/configuration/entities_domains/#domains)

## `service` (Optional)
- The service that the entity will be triggered with.
- Defaults to `trigger`
