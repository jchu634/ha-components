## About
This component is a modified ShadCN button mounts a onClick which triggers a HA service (e.g. automations).\
This is primarily intended for manually activating automations, but can trigger any HA service if the Domain and Service are specified.

## API (Props)
<Callout >
    This only documents props added for triggering HA Services\
    For Button Styling Documentation look [here](https://ui.shadcn.com/docs/components/button)

</Callout>
### `entity`
e.g. `automation.open_curtains`
- A HA entity name

### `description` (Optional)
- Adds a title to the button

### `domain` (Optional)
- The Domain that the entity belongs to.
- Defaults to `automation`
- For more information on domains, see [here](https://www.home-assistant.io/docs/configuration/entities_domains/#domains)

## `service` (Optional)
- The that the entity will be triggered with.
- Defaults to `trigger`
