## API (Props)
### `entity`
e.g. `weather.forecast_home`
- A HA entity name

### `forecastType` (Optional)
- Which time period of forecast to show
- Options: `daily`, `hourly`
- Default: `daily`

<Callout>
Some weather entities in HA do not support some types of forecast
</Callout>

### `refreshIntervalMinutes` (Optional)
- How often to recheck the forecast in minutes.
- Defaults to 1 hour

## `maximumForecastShown` (Optional)
- How many future forecasts to visibly show. (Not inclusive of the current time's weather)
- Defaults to `7`

## `OptionalFeatures` (Optional)
- Which optional statistics to show/hide.
- Options: `rain_chance, temperature, wind_speed, humidity"`
